# botanica

## PROJECT DESCRIPTION

This application allows you to see the number of plants in a garden, see the details of each plant, its location and the care it has received, and allows you to perform maintenance (watering, lighting, pruning, fertilization).

### Links
📌 Live site URL: [here](https://github.com/Tonyva002/proyecto-botanica)

### Built with

- Mysql
- Node.js
- Express
- React.js
- MUI
- Axios
- Docker
- MVVM
- Clean Architecture
- jest
- testing-library
  

### Necessary tool:

- Visual Studio Code (VSC)
- Docker desktop

### Steps to execute:

- Step 1. Download or Clone the repository
  
- Step 2. Unzip the file if you have downloaded it
  
- Step 3. Open in Visual Studio Code (VSC)

- Step 4. In the Visual Studio Code terminal, run:
  
```bash
copy jardin-backend\.env.example jardin-backend\.env
```
- Step 5. Open docker

- Step 6. In the Visual Studio Code terminal, run:

```bash
docker compose up --build
```
- Step 7. In the browser:

```bash
 http://localhost
```
